Final Year Project

Blockchain-Based Voting System

A secure, transparent, and tamper-proof digital voting platform built with modern web technologies

Security First

Cryptographic hashing, digital signatures, and proof-of-work ensure vote integrity and prevent tampering.

Immutable Ledger

Once votes are mined into blocks, they become permanently recorded and cannot be altered.

Transparent

Complete audit trail with real-time validation and tamper detection capabilities.

Technical Architecture

Backend: Python Flask with SQLite database
Blockchain: Custom proof-of-work implementation
Frontend: Bootstrap 5, Chart.js, Font Awesome
Security: HMAC digital signatures, SHA-256 hashing
Deployment: Vercel serverless platform

Key Features

  • User authentication with session management
  • Real-time vote tallying and visualization
  • Admin dashboard with system statistics
  • Comprehensive tamper audit system
  • Activity logging for audit trails
  • Responsive design for all devices
